# ⚡ Generex - Generador de Códigos QR

![Generex Banner](public/og-image.png)

**Generex** es una herramienta web gratuita, rápida y segura para generar códigos QR. Todo el procesamiento se realiza del lado del cliente: tu información **nunca** sale de tu navegador.

✨ **Pruébala aquí:** [generex-qr.vercel.app](https://generex-qr.vercel.app)

## 🚀 Características Principales

- **100% Gratuito:** Sin límites de uso ni suscripciones ocultas.
- **Privacidad Total:** El texto o la URL que introduces se convierte en QR directamente en tu dispositivo, sin pasar por ningún servidor externo.
- **Personalización Visual:** Elige entre diferentes plantillas de color o define tus propios colores de fondo y frente para el QR.
- **Descarga Rápida:** Asigna un nombre personalizado a tu archivo y descárgalo en formato PNG de alta calidad.
- **QR Ilimitados y Permanentes:** Los códigos que generas no caducan, a diferencia de muchos servicios en línea.
